# 测试用例

## 1. DataEntry.tsx 展开/收起功能测试

### 测试目的
验证时间点的展开/收起功能是否正常工作

### 测试步骤
1. 进入应用，配置时间范围，生成多个时间点
2. 点击第一个时间点的展开按钮
3. 验证该时间点的内容是否显示
4. 再次点击该时间点的收起按钮
5. 验证该时间点的内容是否隐藏
6. 点击另一个时间点的展开按钮
7. 验证只有当前点击的时间点内容显示，其他时间点内容隐藏

### 预期结果
- 点击展开按钮后，对应时间点的内容显示
- 点击收起按钮后，对应时间点的内容隐藏
- 同时只能展开一个时间点的内容

## 2. ChartVisualization.tsx 说明文本直接展示测试

### 测试目的
验证图表中的说明文本是否直接显示在图上，无需hover

### 测试步骤
1. 进入应用，配置时间范围和问题
2. 为多个时间点添加说明文本
3. 生成图表
4. 观察图表上的点是否直接显示说明文本
5. 验证说明文本是否清晰可见，不重叠
6. 下载图片，验证说明文本是否也显示在下载的图片中

### 预期结果
- 图表上的每个点都直接显示对应的说明文本
- 说明文本清晰可见，排版合理
- 下载的图片中也包含说明文本
